# Demo App

A simple webpage to demonstrate
[`limited-cache`](https://github.com/spautz/limited-cache/blob/main/packages/limited-cache/).

This in-workspace app is used:
* To demo and document the package from a consumer's perspective
* For CI checks, exercising the package from outside to ensure it works properly
* As a harness for local dev work
